# MIPS — Medicine Intelligence Platform

MIPS is a modern local-first healthcare inventory platform designed to simplify medicine stock management, expiry tracking, demand forecasting, and reporting in one secure and efficient system.

Built for pharmacies, clinics, hospitals, and medical distributors, MIPS helps reduce medicine waste, prevent stock shortages, and improve inventory decisions through real-time insights and predictive analytics.

## ✨ Features

### 📦 Smart Inventory Management

* Manage medicine stock with a clean and searchable inventory system
* Real-time low-stock alerts and availability tracking
* Fast filtering and pagination for large datasets
* Export inventory reports in CSV or PDF format

### 📥 Bulk Import System

* Import medicine data using CSV or XLSX files
* Automatic validation for missing or invalid entries
* Preview uploaded data before saving changes

### ⏳ Expiry & Risk Monitoring

* Track upcoming medicine expiries efficiently
* Reduce wastage with proactive monitoring and alerts
* Identify critical inventory risks instantly

### 📈 Prediction & Forecasting

* AI-assisted demand forecasting and stock prediction
* Visual risk analysis for shortages and overstock situations
* Intelligent recommendations for better inventory planning

### 📊 Analytics Dashboard

* Interactive dashboard with key operational insights
* Trend graphs and inventory performance metrics
* Clear roadmap-style overview for planning and monitoring

### 🎨 User Experience

* Modern red, black, and white design system
* Smooth animations and polished interface interactions
* Light, dark, and auto theme support
* Adjustable text size for improved accessibility

### 🔐 Security

* Secure local authentication system
* JWT-based route and API protection
* Backend validation for secure data handling

## 🚀 Run Locally

```bash
npm install
npm run dev -- --host
```

## 🛠 Build & Validation

```bash
npm run build
npm run test
npm run lint
```
